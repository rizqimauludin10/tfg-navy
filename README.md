# Realtime Tactical Map System / Tactical Floor Game

A web-based interactive map application for monitoring and managing unit positions in real-time.

## 🚀 Features

- 🗺️ Custom map upload and rendering
- 📍 Drag & drop unit positioning
- ⚡ Realtime synchronization using Socket.IO
- 🧭 Movement trail visualization
- 👥 Multi-user collaboration
- 🔍 Unit filtering and search
- 📊 Activity history tracking
- 🔐 Role-based access control (Admin / Superadmin)

## 🛠️ Tech Stack

- Backend: PHP (CodeIgniter 4)
- Frontend: JavaScript, Fabric.js
- Realtime: Socket.IO
- HTTP Client: Axios

## 📌 Overview

This system allows users to visualize and control unit positions on a custom map.  
Each movement is synchronized in real-time across all connected users and recorded as historical data.

Designed for scenarios that require spatial monitoring such as operations tracking, asset management, or tactical visualization.
